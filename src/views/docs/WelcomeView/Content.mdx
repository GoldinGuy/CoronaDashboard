# Welcome
Welcome to **CoronaDash**. This project was aimed at providing an easy way to explore local data and information in light of all the tools available for macro data.
If you're here, you're most likely interested in contributing.

Please visit the ![#Contributing](#Contributing) tab for more information.

# Contributing to the docs

Welcome to the docs.
We'll build stuff out here to explain things.
To add things, copy the example folder, edit the markdown, edit the JavaScript page, and add it to the router.
After doing this, add the route to the navbar.
